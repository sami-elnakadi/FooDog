== Description ==

Ruby is a clean, minimal, responsive and fully customizable WordPress theme that looks awesome on any device. It adjusts automatically to any screen size including tablets and smartphones as well as Retina displays. Theme is built using HTML5, CSS3, Bootstrap 4 and is SEO friendly. This theme can be used for anything so be creative and unleash its full potential in blogging, ecommerce, business and other websites. Along the awesome design this theme is customizable via a simple but yet powerful theme customizer options. Some of the nicest features are social icons, custom logo, translation ready (compatible with WPML plugin) and many more. This theme out-of-the-box plays nicely with WooCommerce, Jetpack, bbPress, BuddyPress, Breadcrumb NavXT, Contact Form 7 and other popular free and premium plugins.

For questions, comments or bug reports, visit Free WP TP support forum (http://freewptp.com/forums/).

== Installation ==

You can install the theme through the WordPress installer under "Themes" > "Add New" by searching for "Ruby".

Alternatively you can download the theme zip file install it from "Themes" > "Add New" > "Upload Theme" or unzip the zip file and move the unzipped contents to the "wp-content/themes" folder of your WordPress installation. You will then be able to activate the theme.

Afterwards you can continue theme setup and customization via WordPress Dashboard > Appearance > Customize. For detailed theme documentation, please visit http://freewptp.com/themes/ruby/doc
